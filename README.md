# 🧠 Memory Card Game

A simple and interactive **Memory Card Game** built using HTML, CSS, and JavaScript. This game helps improve memory skills by challenging players to match pairs of cards within minimum moves and time. 

---

## 🚀 Features

* 🎴 Flip cards to find matching pairs
* ⏱️ Timer to track your performance
* 🔢 Move counter
* 🔄 Restart / Reset game option
* 🎨 Responsive and clean UI
* 🧩 Randomized card positions every time

---

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

---

## 🎮 How to Play

1. Start the game.
2. Click on any card to flip it.
3. Flip another card to find its match.
4. If both cards match, they remain open.
5. If not, they flip back after a short delay.
6. Continue until all pairs are matched.
7. Try to complete the game in minimum moves and time.
